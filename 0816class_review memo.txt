머신러닝
· 지도학습
	- 회귀
	- 분류
· 비지도학습
	- 군집화(clustering) : 비슷한 것 찾아 그룹만들기(K-means, DBSCAN)
	- 변환(transform) : PCA, t-SNE
	- 연관(association) : 서로 관련있는 특성(열)을 찾아주는 머신러닝기법

· 지도학습 vs 비지도학습
	- target값이 있는지 없는지의 차이

· 평가지표
	- 회귀 : MAE, MSE, RMSE, RMLSE, MAPE
	- 분류 : 정확도, 정밀도, 민감도, 특이도, FPRate, f1-score, log-loss

· 데이터 전처리
	- 라벨인코딩 : 컴퓨터가 알아들을 수 있도록 숫자로 변경해줌
	- 원핫인코딩 : 0, 1을 갖는 벡터로 변경, 해당되는 부분을 1로 설정

· 특성자동선택
	- 통계값을 이용한 특성 선택
	- 모델에서 중요하다고 생각하는 특성 선택
	- 모델을 재귀적으로 돌려 그 중 중요하다고 하는 변수 선택

· 비지도학습 
- PCA 
	- 주성분 분석
	- 10개의 변수가 있다면 차원 변환을 통해 10개의 주성분을 만든다.
	- 각각의 주성분은 독립적인 관계를 갖는다.
	- 독립적인 관계라는 것은 서로 전혀 관계가 없다.
	- 주성분1을 만들고, 주성분2을 만들때, 직각인 관계.
